So you want to use jQuery? Here are some things to consider:

  • Download the file, reference it locally.
  • Or, use a Content Delivery Network (CDN)
  • You can also do BOTH - in case the CDN is unavailable.
  • Debugging with the minified version? Ouch. Use HttpContext.Current.IsDebuggingEnabled
  • Put the file before anything that depends on it, but typically as late as you can in the page.
