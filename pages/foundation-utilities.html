jQuery provides a number of handy utility methods. Each of these are handy reusable methods that will save you the time of re-inventing the same common code.

Here are some common uses of utilities:

  • Detect if the certain features are supported
  • Remove whitespace from beginning and end of string
  • Get the current date
  • Merge two arrays together
  • Iterate over an object or array
  • Grep an array
  • Check to see if a DOM element is within another DOM element.

  $.data()
  $.each()
  $.grep()
  $.isFunction()
