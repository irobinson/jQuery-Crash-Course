There are a number of helpful Ajax utilities in jQuery.

  • You can do simple get and post actions, as well as more advanced getJSON, getScript actions.
  • Then there is the most low level - jQuery Ajax
  • For use in all types: there are global ajax event handlers for start, stop, error, and success.
  • For ASMX and WCF goodness: check out Rick Strahl's Service Proxy

A "get" example:

  $('a[href^="pages"]').live('click', function(e){
    e.preventDefault();
    var $this = $(this);
    var targetUrl = $this.attr('href');
    $.get(targetUrl, function(result){
      $('.body').html($(result));
      $('.intro h2').html($this.text());
    });
  });
An "ajax" example from the jquery docs 
  bodyContent = $.ajax({
        url: "script.php",
        global: false,
        type: "POST",
        data: ({id : this.getAttribute('id')}),
        dataType: "html",
        async:false,
        success: function(msg){
           alert(msg);
        }
     }
  ).responseText;
