Animations are also central to a rich user experience. Even the simplest animation, such as showing or hiding an element, can have a significant impact on the user's experience.

Some basic animation concepts:

  • Move an element's location
  • Smoothly change the width/height of an element
  • Hide or show an element
  • Fade an element in or out
  • Animate nearly any CSS property on an element
Common Animation Methods
  $().animate()
  $().fadeIn()
  $().fadeOut()
  $().fadeToggle()
  $().hide()
  $().show()
  $().slideUp()
  $().slideDown()
  $().toggle()
