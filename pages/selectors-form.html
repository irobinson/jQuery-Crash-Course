Intro Basic Attribute Form Extensions
Form Element Selectors

Each of these are essentially shorthand for selecting various types of form elements. Also, these are extensions, not part of the CSS spec.

Form Element Examples
  $(':button')
  $(':checkbox')
  $(':checked') // state based
  $(':disabled') // state based
  $(':input')
  $(':image')
  $(':selected') // state based
  $(':submit')
  $(':text')
