Storing light-weight UI state can be helpful in a number of cases. Two of the most common are described below:

  • When you don't want to tie CSS style to your JavaScript.
  • When you'd like to store values somewhere out of site, to be used later.

Store state as a CSS class

  $('#menu').hover(function() {
    $(this).toggleClass('hover');
  });

Store state as jQuery data

  // set
  $('selector').data('key', 'value');
  
  //get
  $('selector').data('key');
