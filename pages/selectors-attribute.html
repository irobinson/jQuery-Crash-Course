Intro Basic Attribute Form Extensions

Attribute selectors are helpful to narrow down your selection to only elements of a given type that have an attribute set in a specific way.

Attribute Selector Examples
  $(a[href|="value"]) // contains prefix
  $(a[href*="value"]) // contains
  $(a[href~="value"]) // contains word
  $(a[href$="value"]) // ends with
  $(a[href="value"])  // equals
  $(a[href!="value"]) // not equal *
  $(a[href^="value"]) // starts with
  $(a[href])          // has attribute
  $(a[href="value"][id]) // multiple
