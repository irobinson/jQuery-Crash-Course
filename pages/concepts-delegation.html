Event delegation is your friend! It can be used to bind events and not kill your performance (like the "live") event.

  • Binding events is great.
  • You can use shortcuts or the manual "bind" methods.
  • But what if you are dynamically creating elements?
  • Event delegation to the rescue!
  • Never use the "live" method (it can be *much* slower)

Example:

  $("table").delegate("td", "hover", function(){
    $(this).toggleClass("hover");
  });
  
  $(document).delegate("a.external", "click", function() {
    // logic here...
  });
