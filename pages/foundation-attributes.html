When we're talking jQuery, we're also talking HTML. HTML has a DOM with elements that have properties.

Here are a few ways you can work with attributes:

  • Add or remove CSS classes
  • Detect if an element has a particular CSS class
  • Get or set an element's HTML contents
  • Get or set an element's value (form elements primarily)
  • Work with attributes and properties
Attribute Examples
  $('a').addClass('x') // set
  $('a').toggleClass('x') // set
  $('#check').prop('checked') // get
  $('em').attr('title') // get
