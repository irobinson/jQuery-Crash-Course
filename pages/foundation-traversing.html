Traversing is the act of moving from one location to the next through the DOM. Traversing is made pretty simple in jQuery, as the framework provides a number of convenient methods.

Here are some common traversal practices:

  • Get the children of the selected element
  • Find an element within the selected element
  • Get the parent(s) of the selected element
  • Get siblings of the selected element
  • Get the next elements until reaching a specified element
Common Traversal Methods
  • children()
  • closest()
  • find()
  • next(), nextAll(), nextUntil()
  • parent(), parents(), parentsUntil()
  • prev(), prevAll(), prevUntil()
  • siblings()
