Rich user experiences rely heavily on events. The user interacts with the browser and we want to perform actions based on their actions.

Some common event practices:

  • Wait to initalize until everything is done loading
  • Show a modal dialog when a link is clicked
  • Perform validation logic as the user is typing
  • Show a submenu when the user hovers over a parent menu
  • Run some validation logic when a form is submitted
Events

$(document).ready(function(){
  // everything is loaded.
});

$('a').bind('mouseenter mouseleave', function(){
  // add event logic here...
});

  // add event logic here...
}
