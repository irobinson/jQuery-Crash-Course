Which selectors should you use? What are the best for performance? How about maintainability?

  • Keep selectors as simple as possible
  • Selecting an element by its ID will be the fastest.
  • But, classes are general simpler and easier to maintain versus using IDs everywhere. (more reading)
  • Modern browsers can use querySelectorAll for CSS queries.
  • But, jQuery Extensions do not use querySelectorAll. From the docs: "first select some elements using a pure CSS selector, then use .filter()."
  • If it makes sense, use a simple selector, and then do a "find" on it with the more complex selector inside.
